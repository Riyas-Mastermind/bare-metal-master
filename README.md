# ⚡ Bare-Metal Master

Welcome to **Bare-Metal Master**, my professional sandbox for mastering **Embedded C** and low-level hardware programming. This repository serves as a centralized hub for my code implementations, register-level drivers, and technical documentation.

## Project Goals
* **Deep Dive into C:** Mastering pointers, bitwise operations, and memory management.
* **Hardware Abstraction:** Writing peripheral drivers (UART, SPI, I2C, GPIO) from scratch.
* **Optimization:** Learning to write resource-efficient code for constrained environments.
* **Documentation:** Maintaining clear, industry-standard logs of learning milestones.

## 📂 Repository Structure
The project is organized by hardware architecture and learning modules:

* `docs/`: Datasheets, register maps, and my personal study notes.
* `drivers/`: Custom-written drivers for various peripherals.
* `exercises/`: Logic puzzles and bit-manipulation practice.
* `projects/`: Full-scale implementations (e.g., Interrupt-driven systems, Real-time schedulers).

## Tech Stack & Tools
* **Language:** C (C99/C11 standards)
* **Toolchains:** GCC ARM Embedded, STM32ClubeIDE
* **Platforms:** STM32, ARM Cortex-M Series,
* **Utilities:** GDB for debugging, Docker for build environments

## How to Use
1.  **Clone the repo:** `git clone https://github.com/Riyas-Mastermind/bare-metal-master.git`
2.  **Navigate to a module:** Check the `/exercises` folder for getting started with bitwise logic.
3.  **Build:** Most modules include a simple Makefile for quick compilation.
